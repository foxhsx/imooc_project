### 1.什么是 React？

React 是一个用于构建 Web 和原生交互界面的库。它最初是由 Facebook 在 2013 年 5 月 29 日发布的。现在已经迭代到了 18.2.0 的版本。最新的官网地址在这里👉[React](https://react.dev)。

在 React 中我们不直接与 DOM 进行交互，而是通过虚拟 DOM 来进行更改。
