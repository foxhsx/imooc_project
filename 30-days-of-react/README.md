### 课程介绍

30天 React 编程挑战是针对初学者和高级 JS 和 React 开发人员的指南。React 是一个 JS 的库，在这个循序渐进的 30 天编程挑战中，您将学习 React，它是最流行的用户界面 JS 库之一。

React 能做 JS 能做的一切。可用于为网站添加交互性、开发移动端应用、桌面程序和游戏。希望在接下来的30天中大家能学到很多东西，并能提高自己的编程能力和解决问题的能力。
